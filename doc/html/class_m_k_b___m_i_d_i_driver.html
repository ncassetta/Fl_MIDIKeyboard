Detailed Description

The class MKB_MIDIDriver is an object which sends MIDI messages to the computer MIDI ports. It can detect the MIDI ports present on the computer and send them some MIDI channel messages. You can select the port, the channel, the volume, the pan and a default velocity for note messages. The class Fl_MIDIKeyboard inherits from it.
